Git

  • GIT INIT [НАЗВАНИЕ ПРОЕКТА] - Создать новый локальный репозиторий с заданным именем.
  • GIT CLONE [url-адрес] - Скачать репозиторий вместе со всей его историей изменений.
  • GIT STATUS - Перечислить все новые или изменённые файлы, которые нуждаются в фиксации.
  • GIT ADD [ФАЙЛ] - Индексировать указанный файл для последующего коммита.
  • GIT PUSH [УДАЛЁННЫЙ РЕПОЗИТОРИЙ] [ВЕТКА] - Загружает все изменения локальной ветки в удалённый репозиторий.
